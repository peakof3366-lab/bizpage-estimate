    요율 갱신 방법
    1. data.js 파일을 텍스트 편집기(메모장·VS Code)로 열기
    2. 갱신할 도시의 행에서 단가 수치 수정 + "rateDate":"YYYY-MM" 을 현재 월로 변경
    3. 전체 검토 완료 시 RATE_META 의 updated, version, nextReview 도 업데이트
    4. 파일 저장 후 브라우저에서 새로고침 → 변경 내용 즉시 반영

    권장 갱신 주기: 분기별 1회 (3개월마다) · 환율 변동 큰 지역은 월별 확인
