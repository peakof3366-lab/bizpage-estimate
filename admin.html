    📝 실제 계약가 업데이트

    계약이 성사된 실제 항공료·호텔단가·식비를 남겨주세요(전부 채울 필요 없이 아는 항목만 채우면 됩니다) — 목적지별로 모아 기준 요율과 비교해 갱신을 제안합니다. 아래 두 방식 중 하나만 골라서 사용합니다.

    🔒 실제 이용 호텔명은 가격에는 반영되지 않는 관리자 전용 참고 정보입니다 — 고객이 보는 견적서·공개 페이지에는 절대 표시되지 않습니다.

    목적지

    PDF를 올리고 "PDF에서 추출"을 누르면 AI가 문서에서 찾은 항목을 아래에 자동으로 채웁니다(파일 자체는 저장되지 않음 · 텍스트가 있는 PDF만 가능, 스캔 이미지는 지원 안 함). 추출된 값은 직접 수정할 수 없습니다.

    📊 실제 계약 데이터 기반 갱신 제안

    계약 실데이터가 기준 요율과 10% 이상 벌어진 목적지×항목(항공료·호텔·식비, 최소 2건 이상)만 표시됩니다.

    📏 견적 정확도 (실제 대비)

    최근 12개월 계약·실측 기준, 목적지×항목별 견적 대비 실제 오차입니다(표본 2건 이상). 오차가 큰 항목이 위에 옵니다 — 우선 손볼 곳을 알려줍니다.

    목적지항목 중앙 오차범위(p10~p90) 표본
    🏨 실제 이용 호텔 (관리자 전용 참고 정보)

    목적지별 최근 제보 1건만 표시됩니다. 요율(가격)에는 반영되지 않으며, 고객에게는 노출되지 않습니다.

    지역 목적지 시즌 항공료 호텔/박 1인 식비 마진/인 요율 기준월 경과 상태 관리
    요율 갱신 방법

    기본(권장): 위 표에서 목적지의 "편집" 버튼을 눌러 단가를 직접 수정하세요. 저장하는 즉시 실제 견적 계산에 반영되고, 변경 이력에도 자동으로 남습니다.

    1. 관리자 페이지가 아닌 원본 파일을 직접 고쳐야 하는 특수한 경우에만: data.js 파일을 텍스트 편집기로 열어 수정
    2. 이 경우 위 편집 버튼으로 저장한 값(오버라이드)이 우선 적용되므로, 반드시 요율 관리 표에서 해당 목적지의 오버라이드가 없는지 먼저 확인

    권장 갱신 주기: 분기별 1회 (3개월마다) · 환율 변동 큰 지역은 월별 확인

    💱 환율은 매일 새벽 자동으로 확인해, 마지막 저장 시점 대비 5% 이상 움직인 목적지는 자동으로 "확인 권장" 상태가 됩니다(항공료는 인원별 협상 견적이라 자동 반영 대상이 아닙니다 — 배지에 표시되는 % 변동은 참고용 시장 환율 기준).

    이미지는 URL만 입력하는 방식입니다 (파일 업로드 아님) — 새 이미지를 쓰려면 먼저 어딘가에 업로드해서 바로 접근 가능한 이미지 링크(.jpg/.png/.webp로 끝나는 주소)를 확보한 뒤 그 주소를 붙여넣으세요. 각 이미지 항목 아래 표시된 권장 비율·최소 해상도는 실제 그 자리에 어떻게 잘려서 보이는지를 기준으로 계산된 값이니 최대한 맞춰서 준비하면 화질 저하나 어색한 크롭 없이 깔끔하게 노출됩니다. 공통 권장: JPG 또는 WebP 형식, 장당 500KB 이하(로딩 속도), 사진 속 주요 피사체(인물·랜드마크)는 가운데에 배치.
