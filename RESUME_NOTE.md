## ▶ 반드시 지킬 작업 규칙 (과거 사고·합의에서 나온 것)

- **index.html / styles.css / script.js는 절대 전체 재작성 금지** — 섹션 단위 Edit만.
  (과거 전체 재작성이 토큰 한도로 잘려 스타일 통째 유실 사고 있었음.)
- **배포 = `git push origin master`** (프로덕션 자동배포). 계산식 바꾸는 변경은 배포 전
  사용자 승인. additive·계산식 불변 변경은 검증 후 바로 push 가능(정책 합의됨).
- **검증 방법(jsdom)**: `npm install --no-save jsdom` 돼 있음(node_modules는 gitignore).
  테스트 실행 시 `NODE_PATH="<프로젝트경로>/node_modules"` 필요. data.js + dest_currency.js
  + script.js를 **하나의 문자열로 합쳐 한 번에 eval**해야 const 전역이 서로 보임.
- **Vercel Hobby = 서버리스 함수 12개 제한 이미 도달** — 새 API는 새 파일 말고 기존
  파일에 `?action=` 분기로 추가.
- **실측 데이터가 프로덕션 DB에 0건** — 측정기반 작업(P5 신뢰구간 실동작·P6 역검증·
  실측 튜닝)은 계약완료 견적이 쌓여야 의미 있음. 지금은 보류.
- **확정 우선순위 없이 바로 코딩하지 말 것** — 후보 제시하고 사용자 확인 먼저.
