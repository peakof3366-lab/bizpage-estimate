# 링트 웹사이트 샘플

이 프로젝트는 `https://linked.bz/`의 구조와 UX를 참고하여 제작한 한국어 해외 연수 견적 랜딩 페이지 샘플입니다.

## 배포
- 공개 사이트: https://bizpage-estimate.vercel.app
- `master` 브랜치에 push하면 Vercel이 자동으로 재배포합니다.

## 구성
- `index.html` — 메인 페이지 구조
- `styles.css` — 반응형 디자인 스타일
- `script.js` — 간단한 견적 계산기 인터랙션

## 실행 방법
브라우저에서 `index.html` 파일을 열면 사이트를 확인할 수 있습니다.

## 주요 기능
- 1분 견적 CTA와 히어로 섹션
- 지역·인원·기간 기반 예상 견적 계산기
- 2단계 정보 입력으로 구성된 견적 요청 흐름
- 온라인 취합 데이터를 바탕으로 한 객관적 견적 기준표
- 약식 견적서 다운로드 버튼
- 링트의 강점, 서비스 포인트, 회사 소개
- 문의 폼과 연락처 정보

## 개선 제안
- 실제 연수 패키지 / 지역별 요금 데이터를 추가
- 고객 사례, 리뷰, FAQ 섹션 추가
- 서버 기반 문의 접수 또는 상담 신청 폼 연결
